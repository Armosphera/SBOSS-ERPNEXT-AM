## Goal
Create a runnable, installable Frappe custom app at `apps/frappe_armenia` that can be added to a site with `bench --site X install-app frappe_armenia`.

## Files to create/modify
- `apps/frappe_armenia/` (entire Frappe app skeleton, generated by `bench new-app`)
- `apps/frappe_armenia/LICENSE-ARMOSPHERA.md` (copy from repo root)
- `apps/frappe_armenia/pyproject.toml` (add `license = "Armosphera Proprietary"`, add `frappe` and `erpnext` as deps)
- `apps/frappe_armenia/frappe_armenia/hooks.py` (empty app_license = "Armosphera Proprietary")

## Contracts to respect
- **Contract A**: DocTypes we add later are prefixed `AM `.
- **Contract C**: this app may import from `libs/frappe_localization_core` (MIT) but never from `frappe_uae` or `frappe_ai_local`.
- **AGPL-3.0 of upstream ERPNext**: we **link** to it; we do not include any ERPNext source in our repo.

## TDD cycle

### Step 1 — Write failing test
Create `apps/frappe_armenia/tests/test_smoke.py`:
```python
"""
Smoke test: the app installs and is visible to bench.
"""
import unittest
import frappe

class TestAppInstall(unittest.TestCase):
    def test_app_in_installed_apps(self):
        from frappe.installer import _get_installed_apps
        installed = _get_installed_apps(frappe.local.site)
        self.assertIn("frappe_armenia", installed)

    def test_module_loads(self):
        import frappe_armenia
        self.assertTrue(hasattr(frappe_armenia, "__version__"))
```

## Pitfalls
- **`bench new-app` runs interactively** by default. Use all the `--*` flags above to avoid the prompt.
- **License cannot be "AGPL-3.0"** for this app — we are not forking ERPNext, we are linking to it. The custom app is our own work.
- **Do not put any ERPNext source in this app's repo.** We link via `pyproject.toml` deps.
