## Account structure (per Armenian Tax Code Chapter 9)

```
1. Non-current assets         (1000-1999) — Հիմնական միջոցներ
   1.1 Intangible              (1100-1199)
   1.2 Tangible                (1200-1299)
   1.3 Long-term financial     (1300-1399)
   1.4 Deferred tax            (1400-1499)
2. Current assets            (2000-2999) — Շրջանառվող միջոցներ
   2.1 Inventories             (2100-2199)
   2.2 Trade receivables       (2200-2299)
   2.3 Cash and equivalents    (2300-2399)
   2.4 Prepayments and other   (2400-2499)
3. Equity                    (3000-3999) — Սեփական կապիտալ
4. Non-current liabilities   (4000-4999) — Երկարաժամկետ պարտավորություններ
5. Current liabilities       (5000-5999) — Կարճաժամկետ պարտավորություններ
6. Income                    (6000-6999) — Եկամուտներ
7. Expenses                  (7000-7999) — Ծախսեր
8. Off-balance-sheet         (8000-8999) — Հաշվից դուրս
9. Cost of goods sold        (9000-9999) — Ապրանքների ինքնարժեք
```

```python
import unittest
import frappe
from frappe_armenia.coa.install_coa import build_coa_tree, count_by_root_type

class TestArmenianCOA(unittest.TestCase):
    def test_tree_has_all_root_types(self):
        tree = build_coa_tree()
        roots = {a["root_type"] for a in tree}
        self.assertEqual(roots, {"Asset", "Liability", "Equity", "Income", "Expense"})

    def test_account_count_in_range(self):
        tree = build_coa_tree()
        self.assertGreaterEqual(len(tree), 320)
        self.assertLessEqual(len(tree), 400)

    def test_root_type_distribution(self):
        tree = build_coa_tree()
        counts = count_by_root_type(tree)
        self.assertGreaterEqual(counts["Asset"], 100)
        self.assertGreaterEqual(counts["Liability"], 60)
        self.assertGreaterEqual(counts["Equity"], 15)
        self.assertGreaterEqual(counts["Income"], 25)
        self.assertGreaterEqual(counts["Expense"], 80)

    def test_armenian_account_range_mapping(self):
        """Non-current assets must be in 1000-1999 per Armenian Tax Code Ch.9."""
        tree = build_coa_tree()
        nca = [a for a in tree if a["account_name_en"].startswith(("11","12","13","14"))]
        for a in nca:
            self.assertTrue(a["account_number"].isdigit(), f"{a['account_name_en']} has no number")
            num = int(a["account_number"])
            self.assertTrue(1000 <= num <= 1999, f"{a['account_name_en']} num {num} not in 1000-1999")

    def test_bilingual_names_present(self):
        tree = build_coa_tree()
        no_hy = [a for a in tree if not a.get("account_name_hy")]
        self.assertEqual(len(no_hy), 0, f"{len(no_hy)} accounts missing HY name")
```

### Step 3 — Implement `build_coa_tree`
```python
# apps/frappe_armenia/frappe_armenia/coa/install_coa.py
from collections import Counter

# Generated from Armenian Tax Code Chapter 9 (ՀՀ հարկային օրենսգիրք, Գլուխ 9)
# ~350 accounts. For brevity in this task file, only a sample is shown;
# the real implementation must include all 350.
COA_ENTRIES = [
    # (number, name_en, name_hy, root_type, account_type, parent_number)
    ("1000", "Non-current Assets", "Հիմնական միջոցներ", "Asset", None, None),
    ("1100", "Intangible Assets", "Աննյութական ակտիվներ", "Asset", "Fixed Asset", "1000"),
    ("1110", "Software", "Ծրագրային ապահովում", "Asset", "Fixed Asset", "1100"),
    ("1120", "Licenses", "Լիցենզիաներ", "Asset", "Fixed Asset", "1100"),
    # ... (348 more entries)
]

def build_coa_tree() -> list[dict]:
    return [
        {
            "account_number": n,
            "account_name_en": en,
            "account_name_hy": hy,
            "root_type": rt,
            "account_type": at,
            "parent_account_number": pn,
        }
        for (n, en, hy, rt, at, pn) in COA_ENTRIES
    ]

def count_by_root_type(tree: list[dict]) -> dict[str, int]:
    return dict(Counter(a["root_type"] for a in tree))

def seed_armenian_coa(company: str) -> int:
    """Insert all accounts from build_coa_tree() under the given company.
    Returns the number of accounts created.
    """
    import frappe
    tree = build_coa_tree()
    # Two-pass: parents first, then children
    by_num = {}
    for entry in tree:
        doc = frappe.new_doc("Account")
        doc.company = company
        doc.account_number = entry["account_number"]
        doc.account_name = entry["account_name_en"]
        # Store HY name in a custom field (see W1-T05)
        doc.account_name_hy = entry["account_name_hy"]
        doc.root_type = entry["root_type"]
        doc.account_type = entry["account_type"] or ""
        if entry["parent_account_number"]:
            doc.parent_account = by_num[entry["parent_account_number"]]
        doc.insert(ignore_permissions=True)
        by_num[entry["account_number"]] = doc.name
    return len(tree)

def seed_armenian_coa_for_new_company(doc, method=None):
    """Hooks after Company on_update: seed COA if it's a new company."""
    if doc.is_new():
        seed_armenian_coa(doc.name)
```

## Pitfalls
- **Account number uniqueness**: ERPNext requires `(company, account_number)` to be unique. If you re-run the seed, it will throw. Use `frappe.db.exists` to skip existing.
- **`account_name` length**: Armenian text can be longer than English. Test with a real Armenian account name like "Հիմնական միջոցների հաշվային արժեք".
- **Two-pass insertion**: parents must exist before children. The example code does this.
- **Tax Code mapping**: at this stage, do not create tax templates. That's W1.b's job (W1-T10..T17).
