### Step 3 — Implement
```python
# libs/frappe_localization_core/frappe_localization_core/number_to_words.py
from __future__ import annotations
from decimal import Decimal
from num2words import num2words

SUPPORTED = ("en", "hy", "ar", "ru")

def number_to_words(value: Decimal, lang: str) -> str:
    """Convert a Decimal to its word representation in the given language.

    Args:
        value: any Decimal (integer, fractional, negative).
        lang:  one of "en", "hy", "ar", "ru". Raises ValueError otherwise.

    Returns:
        The number in words. For negative numbers, the word for "minus"
        is prepended. Fractional part is spelled digit-by-digit after
        "point".
    """
    if lang not in SUPPORTED:
        raise ValueError(f"Unsupported language: {lang!r}. Supported: {SUPPORTED}")
    if not isinstance(value, Decimal):
        value = Decimal(str(value))

    sign = ""
    if value < 0:
        sign = {"en": "minus ", "hy": "մինուս ", "ar": "ناقص ", "ru": "минус "}[lang]
        value = -value

    int_part = int(value)
    frac = value - int_part
    if frac == 0:
        words = num2words(int_part, lang=lang)
    else:
        # Spell fractional digits one-by-one (universal convention)
        digits = str(frac).split(".")[1]  # e.g. "34" from 0.34
        int_words = num2words(int_part, lang=lang) if int_part else num2words(0, lang=lang)
        point_word = {"en": "point", "hy": "կետ", "ar": "فاصلة", "ru": "точка"}[lang]
        frac_words = " ".join(num2words(int(d), lang=lang) for d in digits)
        words = f"{int_words} {point_word} {frac_words}"
    return f"{sign}{words}"
```

## Pitfalls
- **num2words HY support**: check first; it has been historically incomplete for Armenian. If absent, scope this task to en/ru/ar and open a follow-up task to ship a small HY converter.
- **num2words locale codes**: for Arabic it uses `"ar"` for Western-Arabic numeral words. For Arabic-Indic numerals (٠١٢٣), use `"ar_001"` or post-process.
- **Decimal vs float**: `Decimal("0.1") + Decimal("0.2")` = `0.3` exactly; `0.1 + 0.2` = `0.30000000000000004`. Use `Decimal` always.
- **The Armenian word for "minus" is "մինուս"** (a transliteration). This is what Armenian bank forms use.
